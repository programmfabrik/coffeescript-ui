# coffeescript-ui
Coffeescript User Interface System (CUI)

# install

    git clone https://github.com/programmfabrik/coffeescript-ui.git coffeescript-ui
    git submodule init
    git submodule update
    make

In **build** the needed files can be found after the "make".
